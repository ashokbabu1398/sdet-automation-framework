# 🚀 SDET Automation Framework

## 🔧 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- JSON (Test Data Handling)

## 📁 Project Structure
src/
 ├── main/java/base        → WebDriver setup
 ├── main/java/pages       → Page Object Model (POM)
 ├── test/java/tests       → Test cases

testdata/                  → External test data

## 🚀 Features
✔ Page Object Model (POM) design  
✔ Data-driven testing using JSON  
✔ Scalable automation framework  
✔ Reusable test components  
✔ Ready for CI/CD integration  

## ▶️ How to Run
1. Clone the repository:
   git clone https://github.com/ashokbabu1398/sdet-automation-framework.git

2. Navigate to project:
   cd sdet-automation-framework

3. Run tests:
   mvn test
